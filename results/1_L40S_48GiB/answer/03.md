# Prompt
C++로 빠른 푸리에 변환(FFT)을 구현해봐.

# Answer
물론입니다! C++로 **빠른 푸리에 변환**(Fast Fourier Transform, FFT)을 구현하는 것은 신호 처리, 이미지 처리, 데이터 분석 등에서 매우 유용합니다. 아래는 **Cooley-Tukey 알고리즘**을 기반으로 한 **재귀적 FFT 구현** 예제입니다.

---

## ✅ C++로 구현한 FFT (Cooley-Tukey 알고리즘)

```cpp
#include <iostream>
#include <vector>
#include <complex>
#include <cmath>

using namespace std;

const double PI = acos(-1);

// FFT 함수: 재귀적으로 구현
void fft(vector<complex<double>>& a, bool invert) {
    int n = a.size();
    if (n <= 1) return;

    // 짝수 인덱스와 홀수 인덱스로 분할
    vector<complex<double>> even(n / 2);
    vector<complex<double>> odd(n / 2);

    for (int i = 0; i < n / 2; ++i) {
        even[i] = a[2 * i];
        odd[i] = a[2 * i + 1];
    }

    fft(even, invert);
    fft(odd, invert);

    // 회전 인자 (w)
    double angle = 2 * PI / n * (invert ? -1 : 1);
    complex<double> w(1), wn(cos(angle), sin(angle));

    for (int i = 0; i < n / 2; ++i) {
        a[i] = even[i] + w * odd[i];
        a[i + n / 2] = even[i] - w * odd[i];
        w *= wn;
    }
}

// FFT를 사용하여 두 다항식의 곱셈을 계산 (FFT로 convolution 계산)
vector<long long> multiply(const vector<int>& a, const vector<int>& b) {
    vector<complex<double>> fa(a.begin(), a.end());
    vector<complex<double>> fb(b.begin(), b.end());

    int n = 1;
    while (n < a.size() + b.size()) n <<= 1;
    fa.resize(n);
    fb.resize(n);

    fft(fa, false);
    fft(fb, false);

    for (int i = 0; i < n; ++i) {
        fa[i] *= fb[i];
    }

    fft(fa, true);

    vector<long long> result(n);
    for (int i = 0; i < n; ++i) {
        result[i] = round(fa[i].real());
    }

    return result;
}

// 예시: 두 다항식 곱셈
int main() {
    vector<int> a = {1, 2, 3};  // 1 + 2x + 3x^2
    vector<int> b = {4, 5, 6};  // 4 + 5x + 6x^2

    auto result = multiply(a, b);

    cout << "결과: ";
    for (int i = 0; i < result.size(); ++i) {
        cout << result[i] << " ";
    }
    cout << endl;

    return 0;
}
```

---

## 📌 설명:

- `fft` 함수는 **Cooley-Tukey FFT 알고리즘**을 사용하여 복소수 배열을 FFT로 변환합니다.
- `invert` 파라미터는 역변환(IfFT)을 위한 플래그입니다.
- `multiply` 함수는 두 다항식의 곱을 FFT를 이용해 계산합니다.
- `complex<double>`는 `<complex>` 헤더에서 제공됩니다.

```cpp
a = [1, 2, 3]
```

```
결과: 4 13 28 27 18
```

## 🚀 추가 팁:

- 실제 응용에서는 **버터플라이 구조**를 사용한 **반복적 FFT**가 더 효율적일 수 있습니다.
- `n`이 2의 거듭제곱이 되도록 padding을 해주는 것이 좋습니다.
- `complex` 대신 `std::valarray` 또는 `Eigen` 같은 라이브러리를 사용하면 성능 향상 가능.
