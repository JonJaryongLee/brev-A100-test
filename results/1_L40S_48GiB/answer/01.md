# Prompt
파이썬에서 리스트를 정렬하는 함수를 작성해줘.

# Answer
물론입니다! 파이썬에서 리스트를 정렬하는 방법은 여러 가지가 있지만, 가장 기본적인 방법으로 `sorted()` 함수와 `list.sort()` 메서드를 사용할 수 있습니다. 아래에 각각의 예시를 드릴게요.

---

### 1. `sorted()` 함수 사용 (원본 리스트 변경 없음)

```python
# 원본 리스트를 변경하지 않고 정렬된 새로운 리스트를 반환
my_list = [3, 1, 4, 1, 5, 9, 2, 6]
sorted_list = sorted(my_list)
print("원본:", my_list)
print("정렬된 리스트:", sorted_list)
```

**출력:**
```
원본: [3, 1, 4, 1, 5, 9, 2, 6]
정렬된 리스트: [1, 1, 2, 3, 4, 5, 6, 9]
```

---

### 2. `list.sort()` 메서드 사용 (원본 리스트 변경)

```python
# 원본 리스트를 직접 정렬
my_list = [3, 1, 4, 1, 5, 9, 2, 6]
my_list.sort()
print("정렬된 리스트:", my_list)
```

**출력:**
```
정렬된 리스트: [1, 1, 2, 3, 4, 5, 6, 9]
```

---

### 3. 내림차순 정렬

```python
# 내림차순 정렬
my_list = [3, 1, 4, 1, 5, 9, 2, 6]
sorted_desc = sorted(my_list, reverse=True)
print("내림차순:", sorted_desc)
```

**출력:**
```
내림차순: [9, 6, 5, 4, 3, 2, 1, 1]
```

---

### 4. 문자열 리스트 정렬

```python
# 문자열 리스트 정렬
fruits = ["banana", "apple", "cherry", "date"]
sorted_fruits = sorted(fruits)
print("정렬된 과일:", sorted_fruits)
```

**출력:**
```
정렬된 과일: ['apple', 'banana', 'cherry', 'date']
```

---

### 5. 사용자 정의 정렬 기준 (예: 길이 기준)

```python
# 문자열을 길이 순으로 정렬
words = ["python", "java", "c", "javascript"]
sorted_by_length = sorted(words, key=len)
print("길이 순 정렬:", sorted_by_length)
```

**출력:**
```
길이 순 정렬: ['c', 'java', 'python', 'javascript']
```
