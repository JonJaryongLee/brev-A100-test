# Answer
물론입니다! 아래는 간단한 **FastAPI** 애플리케이션 예제 코드입니다. 이 애플리케이션은 `/` 경로에 GET 요청을 보내면 "Hello World"를 반환하고, `/users/{user_id}` 경로에 GET 요청을 보내면 해당 사용자 정보를 반환합니다.

### ✅ 요구사항:
- Python 3.7 이상
- `fastapi` 및 `uvicorn` 설치 필요

```bash
pip install fastapi uvicorn
```

---

### 📄 예제 코드 (`main.py`):

```python
from fastapi import FastAPI

app = FastAPI()

# 메인 페이지
@app.get("/")
def read_root():
    return {"message": "Hello World"}

# 사용자 정보 조회
@app.get("/users/{user_id}")
def read_user(user_id: int):
    return {"user_id": user_id, "name": f"User {user_id}"}
```

---

### 🚀 실행 방법:

터미널에서 다음 명령어를 입력해주세요:

```bash
uvicorn main:app --reload
```

- `--reload` 옵션은 코드 변경 시 자동으로 서버 재시작해줍니다.
- 서버가 실행되면 브라우저에서 [http://127.0.0.1:8000](http://127.0.0.1:8000)로 접속하면 "Hello World"가 나옵니다.
- [http://127.0.0.1:8000/users/123](http://127.0.0.1:8000/users/123)처럼 URL에 ID를 넣으면 해당 사용자 정보가 나옵니다.
