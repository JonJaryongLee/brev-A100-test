# Prompt
로그인 기능을 테스트하기 위한 유닛 테스트 코드를 작성해줘.

# Answer
물론입니다! 아래는 Python의 `unittest` 라이브러리를 사용하여 로그인 기능을 테스트하는 예제 코드입니다. 이 예제에서는 간단한 로그인 로직을 가정하고, 유효한 자격증명과 유효하지 않은 자격증명에 대한 테스트를 진행합니다.

---

### ✅ 가정 사항:

- `login(username, password)` 함수가 주어졌다고 가정합니다.
- 이 함수는 `username`과 `password`를 받아서, 정확한 자격증명이면 `True`, 그렇지 않으면 `False`를 반환합니다.
- 실제 로그인 로직은 데이터베이스나 외부 API와 연결되어 있을 수 있습니다.

---

### 📁 예시 코드 구조

```python
# login.py
def login(username, password):
    # 예시: 정적 사용자 데이터
    valid_users = {
        "admin": "1234",
        "user1": "pass123"
    }

    if username in valid_users and valid_users[username] == password:
        return True
    return False
```

---

### 🧪 유닛 테스트 코드 (`test_login.py`)

```python
import unittest
from login import login

class TestLogin(unittest.TestCase):

    def test_valid_login(self):
        # 유효한 자격증명으로 로그인 시도
        result = login("admin", "1234")
        self.assertTrue(result)

    def test_invalid_username(self):
        # 존재하지 않는 사용자 이름
        result = login("nonexistent", "1234")
        self.assertFalse(result)

    def test_invalid_password(self):
        # 올바른 사용자 이름이지만 잘못된 비밀번호
        result = login("admin", "wrongpass")
        self.assertFalse(result)

    def test_empty_credentials(self):
        # 빈 자격증명
        result = login("", "")
        self.assertFalse(result)

if __name__ == '__main__':
    unittest.main()
```
