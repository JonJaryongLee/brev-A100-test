# Prompt
로그인 기능을 테스트하기 위한 유닛 테스트 코드를 작성해줘.

# Answer
물론입니다! 아래는 Python의 `unittest` 라이브러리를 사용하여 로그인 기능을 테스트하는 예제 코드입니다. 이 예제에서는 간단한 로그인 로직을 가정하고, 유효한 자격 증명과 유효하지 않은 자격 증명에 대한 테스트를 진행합니다.

---

### ✅ 가정 사항:

- `login(username, password)` 함수가 존재합니다.
- 이 함수는 사용자 이름과 비밀번호를 받아서 로그인 성공/실패 여부를 반환합니다.
- 실제 구현은 DB나 외부 API와 연동될 수 있지만, 여기서는 가상의 로직으로 처리하겠습니다.

---

### 📁 예제 코드 구조

```
project/
│
├── auth.py              # 로그인 로직이 있는 파일
└── test_auth.py         # 유닛 테스트 파일
```

---

## 1. `auth.py` – 로그인 로직

```python
# auth.py

def login(username, password):
    # 예시: 정적 사용자 데이터
    valid_users = {
        "admin": "password123",
        "user1": "mypassword"
    }

    if username in valid_users and valid_users[username] == password:
        return True
    return False
```

---

## 2. `test_auth.py` – 유닛 테스트

```python
# test_auth.py

import unittest
from auth import login

class TestLogin(unittest.TestCase):

    def test_valid_login(self):
        # 유효한 자격 증명으로 로그인 시도
        result = login("admin", "password123")
        self.assertTrue(result)

    def test_invalid_username(self):
        # 존재하지 않는 사용자 이름
        result = login("nonexistent", "password123")
        self.assertFalse(result)

    def test_invalid_password(self):
        # 올바른 사용자 이름이지만 잘못된 비밀번호
        result = login("admin", "wrongpassword")
        self.assertFalse(result)

    def test_empty_credentials(self):
        # 빈 값으로 로그인 시도
        result = login("", "")
        self.assertFalse(result)

if __name__ == '__main__':
    unittest.main()
```

## 🔧 실행 방법

터미널에서 다음 명령어로 테스트를 실행할 수 있습니다:

```bash
python -m unittest test_auth.py
```

또는 `-v` 옵션을 붙여서 상세 출력도 가능합니다:

```bash
python -m unittest test_auth.py -v
```
