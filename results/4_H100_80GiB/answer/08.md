# Prompt
SQL에서 중복 데이터를 찾는 쿼리를 작성해줘.

# Answer
SQL에서 중복 데이터를 찾기 위해서는 `GROUP BY`와 `HAVING` 절을 사용하거나, `ROW_NUMBER()` 함수를 활용하는 방법이 있습니다. 아래에 몇 가지 예시를 들어드리겠습니다.

---

### ✅ 1. 특정 컬럼 기준으로 중복 데이터 찾기

예를 들어, `users` 테이블에서 `email` 컬럼이 중복되는 행을 찾고 싶다면:

```sql
SELECT email, COUNT(*) AS cnt
FROM users
GROUP BY email
HAVING COUNT(*) > 1;
```

이 쿼리는 `email`이 중복된 경우만 출력하며, 그 개수도 함께 보여줍니다.

---

### ✅ 2. 모든 컬럼을 기준으로 중복 데이터 찾기

모든 컬럼이 동일한 행이 있는지 확인하려면:

```sql
SELECT *
FROM users
GROUP BY name, email, age
HAVING COUNT(*) > 1;
```

---

### ✅ 3. ROW_NUMBER()를 사용하여 중복 데이터 찾기 (중복된 행 모두 보기)

```sql
SELECT *
FROM (
    SELECT *,
           ROW_NUMBER() OVER (PARTITION BY email ORDER BY id) AS rn
    FROM users
) t
WHERE rn > 1;
```

이 쿼리는 `email`이 중복되는 모든 행을 보여주며, 첫 번째 행은 제외하고 나머지를 출력합니다.

---

### ✅ 4. 중복된 행을 삭제하기 전에 확인할 때 사용

```sql
SELECT email, COUNT(*) AS duplicate_count
FROM users
GROUP BY email
HAVING COUNT(*) > 1
ORDER BY duplicate_count DESC;
```

---

필요한 구조나 테이블 이름이 있다면 더 구체적인 쿼리를 도와드릴 수 있어요!
