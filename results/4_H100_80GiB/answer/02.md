# Prompt
파이썬 데코레이터의 사용법과 예시를 설명해줘.

# Answer
파이썬 데코레이터(decorator)는 함수나 클래스에 추가적인 기능을 쉽게 붙일 수 있도록 해주는 문법입니다.  
데코레이터는 **함수를 입력으로 받아서 새로운 함수를 반환하는 함수**입니다.

---

## 🧠 기본 개념

```python
@decorator
def my_function():
    pass
```

위 코드는 아래와 동일합니다:

```python
def my_function():
    pass

my_function = decorator(my_function)
```

즉, `@decorator`는 `decorator(my_function)`을 줄여 쓴 것입니다.

---

## ✅ 간단한 데코레이터 예제

### 1. 기본적인 데코레이터 만들기

```python
def my_decorator(func):
    def wrapper():
        print("함수 실행 전")
        func()
        print("함수 실행 후")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()
```

**출력:**
```
함수 실행 전
Hello!
함수 실행 후
```

## 🔄 데코레이터에 인자 전달하기

함수에 인자를 받는 경우, `wrapper`도 인자를 받아야 합니다.

```python
def my_decorator(func):
    def wrapper(*args, **kwargs):
        print("함수 실행 전")
        result = func(*args, **kwargs)
        print("함수 실행 후")
        return result
    return wrapper

@my_decorator
def greet(name):
    print(f"Hello, {name}!")

greet("Alice")
```

**출력:**
```
함수 실행 전
Hello, Alice!
함수 실행 후
```

## ⏱️ 실행 시간 측정 데코레이터 예제

```python
import time

def timer(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"{func.__name__} 실행 시간: {end - start:.4f}초")
        return result
    return wrapper

@timer
def slow_function():
    time.sleep(1)
    print("완료")

slow_function()
```

**출력:**
```
완료
slow_function 실행 시간: 1.0032초
```

## 🧪 클래스형 데코레이터

클래스를 사용해서 데코레이터를 구현할 수도 있습니다.

```python
class MyDecorator:
    def __init__(self, func):
        self.func = func

    def __call__(self, *args, **kwargs):
        print("클래스형 데코레이터 시작")
        result = self.func(*args, **kwargs)
        print("클래스형 데코레이터 종료")
        return result

@MyDecorator
def hello():
    print("안녕하세요!")

hello()
```

**출력:**
```
클래스형 데코레이터 시작
안녕하세요!
클래스형 데코레이터 종료
```

## 🧩 여러 데코레이터 함께 사용

```python
def bold(func):
    def wrapper():
        return "<b>" + func() + "</b>"
    return wrapper

def italic(func):
    def wrapper():
        return "<i>" + func() + "</i>"
    return wrapper

@bold
@italic
def get_text():
    return "Hello"

print(get_text())
```

**출력:**
```
<b><i>Hello</i></b>
```

> 데코레이터는 위에서 아래로 적용됩니다 (`@italic` → `@bold`)

---

## 📌 요약

- 데코레이터는 함수를 감싸서 기능을 추가하거나 수정할 때 사용합니다.
- `@decorator`는 `decorator(func)`의 syntactic sugar입니다.
- `*args`, `**kwargs`를 사용하면 모든 함수에 적용 가능합니다.
- 여러 데코레이터를 함께 사용할 수 있으며, 순서가 중요합니다.
