# Prompt
로그인 기능을 테스트하기 위한 유닛 테스트 코드를 작성해줘.

# Answer
물론입니다! 아래는 Python의 `unittest` 라이브러리를 사용하여 로그인 기능을 테스트하는 예제 코드입니다. 이 예제에서는 간단한 로그인 로직을 가정하고, 사용자 이름과 비밀번호가 일치하는지 확인하는 테스트를 진행합니다.

---

### ✅ 가정 조건:

- `login(username, password)` 함수가 존재합니다.
- 이 함수는 사용자 이름과 비밀번호가 정확히 일치하면 `True`, 그렇지 않으면 `False`를 반환합니다.
- 사용자 정보는 메모리에 저장되어 있다고 가정합니다 (실제 애플리케이션에서는 DB 사용).

---

### 📁 파일 구조

```
project/
│
├── auth.py          # 로그인 로직
└── test_auth.py     # 유닛 테스트
```

---

## 1. `auth.py` – 로그인 로직

```python
# auth.py

def login(username, password):
    # 간단한 사용자 데이터 (실제에는 DB에서 가져옴)
    users = {
        "admin": "password123",
        "user1": "mypassword"
    }

    if username in users and users[username] == password:
        return True
    return False
```

---

## 2. `test_auth.py` – 유닛 테스트

```python
# test_auth.py

import unittest
from auth import login

class TestLogin(unittest.TestCase):

    def test_valid_login(self):
        # 유효한 로그인 테스트
        result = login("admin", "password123")
        self.assertTrue(result)

    def test_invalid_username(self):
        # 존재하지 않는 사용자 테스트
        result = login("nonexistent", "password123")
        self.assertFalse(result)

    def test_invalid_password(self):
        # 잘못된 비밀번호 테스트
        result = login("admin", "wrongpassword")
        self.assertFalse(result)

    def test_empty_credentials(self):
        # 빈 값 입력 테스트
        result = login("", "")
        self.assertFalse(result)

if __name__ == '__main__':
    unittest.main()
```
