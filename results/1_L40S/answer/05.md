# Prompt
간단한 FastAPI 애플리케이션 예제 코드를 짜줘.

# Answer
물론입니다! 아래는 간단한 **FastAPI** 애플리케이션 예제 코드입니다. 이 애플리케이션은 `/` 경로에 GET 요청을 보내면 "Hello World"를 반환하고, `/users/{user_id}` 경로에 GET 요청을 보내면 해당 사용자 정보를 반환합니다.

### ✅ 예제 코드 (main.py)

```python
from fastapi import FastAPI

app = FastAPI()

# 루트 경로
@app.get("/")
def read_root():
    return {"message": "Hello World"}

# 사용자 정보 조회
@app.get("/users/{user_id}")
def read_user(user_id: int):
    return {"user_id": user_id, "name": f"User {user_id}"}
```

---

### 🚀 실행 방법

1. **FastAPI 설치**:
   ```bash
   pip install fastapi uvicorn
   ```

2. **파일 저장**: 위 코드를 `main.py`로 저장합니다.

3. **서버 실행**:
   ```bash
   uvicorn main:app --reload
   ```

4. **브라우저에서 확인**:
   - `http://127.0.0.1:8000/` → `"Hello World"`
   - `http://127.0.0.1:8000/users/123` → `{"user_id": 123, "name": "User 123"}`
