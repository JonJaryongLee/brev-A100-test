# Prompt
파이썬 데코레이터의 사용법과 예시를 설명해줘.

# Answer
파이썬 데코레이터(decorator)는 함수나 클래스에 추가적인 기능을 쉽게 붙일 수 있도록 해주는 문법입니다.  
데코레이터는 **함수를 입력으로 받아서 새로운 함수를 반환하는 함수**입니다.

---

## 🧠 기본 개념

```python
@decorator
def my_function():
    pass
```

위 코드는 아래와 동일합니다:

```python
def my_function():
    pass

my_function = decorator(my_function)
```

즉, `@decorator`는 `decorator(my_function)`을 줄여 쓴 것입니다.

---

## ✅ 간단한 데코레이터 예제

### 1. 기본적인 데코레이터 만들기

```python
def my_decorator(func):
    def wrapper():
        print("함수 실행 전")
        func()
        print("함수 실행 후")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()
```

**출력:**
```
함수 실행 전
Hello!
함수 실행 후
```

## 🧪 더 실용적인 예제들

### 2. 인자를 받는 함수에 데코레이터 적용

```python
def repeat(times):
    def decorator(func):
        def wrapper(*args, **kwargs):
            for _ in range(times):
                result = func(*args, **kwargs)
            return result
        return wrapper
    return decorator

@repeat(times=3)
def greet(name):
    print(f"Hello, {name}")

greet("Alice")
```

**출력:**
```
Hello, Alice
Hello, Alice
Hello, Alice
```

---

### 3. 시간 측정 데코레이터

```python
import time

def timer(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"{func.__name__} took {end - start:.4f} seconds")
        return result
    return wrapper

@timer
def slow_function():
    time.sleep(1)
    return "Done"

slow_function()
```

**출력:**
```
slow_function took 1.0001 seconds
```

### 4. `functools.wraps` 사용하기 (함수 메타데이터 유지)

```python
from functools import wraps

def my_decorator(func):
    @wraps(func)  # 원래 함수 정보를 유지
    def wrapper(*args, **kwargs):
        print("Before function call")
        result = func(*args, **kwargs)
        print("After function call")
        return result
    return wrapper

@my_decorator
def example_func():
    """This is an example function."""
    return "Hello!"

print(example_func.__name__)  # 출력: example_func
print(example_func.__doc__)   # 출력: This is an example function.
```

## 🧩 데코레이터 활용 예시

### 5. 로그 기록 데코레이터

```python
import logging

logging.basicConfig(level=logging.INFO)

def log_calls(func):
    def wrapper(*args, **kwargs):
        logging.info(f"Calling {func.__name__}")
        result = func(*args, **kwargs)
        logging.info(f"{func.__name__} returned {result}")
        return result
    return wrapper

@log_calls
def add(a, b):
    return a + b

add(3, 5)
```

**출력:**
```
INFO:root:Calling add
INFO:root:add returned 8
```

## 🔚 요약

- 데코레이터는 함수를 감싸서 기능을 추가할 때 사용합니다.
- `@decorator` 문법은 `decorator(func)`의 축약형입니다.
- `functools.wraps`는 원래 함수의 이름과 문서를 유지하는 데 유용합니다.
- 데코레이터는 로깅, 타이머, 권한 체크, 캐싱 등 다양한 용도로 활용됩니다.
