# Prompt
로그인 기능을 테스트하기 위한 유닛 테스트 코드를 작성해줘.

# Answer
물론입니다! 아래는 Python의 `unittest` 라이브러리를 사용하여 로그인 기능을 테스트하는 예제 코드입니다. 이 예제에서는 간단한 로그인 로직을 가정하고, 유효한 자격증명과 유효하지 않은 자격증명에 대한 테스트를 진행합니다.

---

### ✅ 가정 사항:

- `login(username, password)` 함수가 주어졌다고 가정합니다.
- 이 함수는 사용자 이름과 비밀번호를 받아서, 유효하면 `True`, 그렇지 않으면 `False`를 반환합니다.
- 실제 구현은 DB나 외부 API와 연동될 수 있지만, 여기서는 단순히 메모리 상의 데이터를 사용합니다.

---

### 📁 예시 코드 구조

```
login_system.py        # 로그인 로직이 있는 파일
test_login.py          # 유닛 테스트 코드
```

---

## 1. `login_system.py`

```python
# login_system.py

def login(username, password):
    # 간단한 mock 데이터
    valid_users = {
        "admin": "password123",
        "user1": "mypassword"
    }

    if username in valid_users and valid_users[username] == password:
        return True
    return False
```

---

## 2. `test_login.py`

```python
# test_login.py

import unittest
from login_system import login

class TestLogin(unittest.TestCase):

    def test_valid_login(self):
        # 유효한 자격증명으로 로그인 시도
        result = login("admin", "password123")
        self.assertTrue(result)

    def test_invalid_username(self):
        # 존재하지 않는 사용자 이름
        result = login("nonexistent", "password123")
        self.assertFalse(result)

    def test_invalid_password(self):
        # 올바른 사용자 이름이지만 잘못된 비밀번호
        result = login("admin", "wrongpassword")
        self.assertFalse(result)

    def test_empty_credentials(self):
        # 빈 자격증명
        result = login("", "")
        self.assertFalse(result)

if __name__ == '__main__':
    unittest.main()
```
