# Prompt
파이썬 데코레이터의 사용법과 예시를 설명해줘.

# Answer
파이썬 데코레이터(decorator)는 함수나 클래스에 추가적인 기능을 쉽게 붙일 수 있도록 해주는 문법입니다.  
데코레이터는 **함수를 입력으로 받아서 새로운 함수를 반환하는 함수**입니다.

---

## 🧠 기본 개념

```python
@decorator
def my_function():
    pass
```

위 코드는 아래와 동일합니다:

```python
def my_function():
    pass

my_function = decorator(my_function)
```

즉, `@decorator`는 `decorator(my_function)`을 줄여 쓴 것입니다.

---

## ✅ 간단한 데코레이터 예제

### 1. 기본적인 데코레이터 만들기

```python
def my_decorator(func):
    def wrapper():
        print("함수 실행 전")
        func()
        print("함수 실행 후")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()
```

**출력:**
```
함수 실행 전
Hello!
함수 실행 후
```

## 🔄 데코레이터에 인자 전달하기

함수에 인자를 받는 경우, `wrapper`도 인자를 받아야 합니다.

```python
def my_decorator(func):
    def wrapper(*args, **kwargs):
        print("함수 실행 전")
        result = func(*args, **kwargs)
        print("함수 실행 후")
        return result
    return wrapper

@my_decorator
def greet(name):
    print(f"Hello, {name}!")

greet("Alice")
```

**출력:**
```
함수 실행 전
Hello, Alice!
함수 실행 후
```

## ⏱️ 시간 측정 데코레이터 예제

```python
import time

def timer(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"{func.__name__} 실행 시간: {end - start:.4f}초")
        return result
    return wrapper

@timer
def slow_function():
    time.sleep(1)
    print("완료")

slow_function()
```

**출력:**
```
완료
slow_function 실행 시간: 1.0032초
```

## 🧪 클래스형 데코레이터

클래스를 사용해서 데코레이터를 구현할 수도 있습니다.

```python
class MyDecorator:
    def __init__(self, func):
        self.func = func

    def __call__(self, *args, **kwargs):
        print("클래스형 데코레이터 시작")
        result = self.func(*args, **kwargs)
        print("클래스형 데코레이터 종료")
        return result

@MyDecorator
def my_func():
    print("함수 실행 중")

my_func()
```

**출력:**
```
클래스형 데코레이터 시작
함수 실행 중
클래스형 데코레이터 종료
```

## 🛠️ 실제 활용 예시: 로깅 데코레이터

```python
import logging

def log_calls(func):
    def wrapper(*args, **kwargs):
        logging.info(f"Calling {func.__name__} with args={args}, kwargs={kwargs}")
        result = func(*args, **kwargs)
        logging.info(f"{func.__name__} returned {result}")
        return result
    return wrapper

@log_calls
def add(a, b):
    return a + b

add(3, 5)
```

이렇게 하면 `add` 함수가 호출될 때 로그가 남습니다.

## 🔁 데코레이터를 여러 개 사용하기

```python
@decorator1
@decorator2
def my_function():
    pass
```

이는 다음과 같습니다:

```python
my_function = decorator1(decorator2(my_function))
```

즉, 위에서부터 아래로 적용됩니다.

---

## ✅ 요약

- 데코레이터는 함수를 감싸서 기능을 추가하거나 수정할 수 있음.
- `@` 문법을 사용하여 쉽게 적용 가능.
- `wrapper` 함수를 통해 원래 함수의 동작을 확장하거나 변경 가능.
- 유용한 패턴: 타이머, 로깅, 권한 체크 등.
