# Prompt
파이썬에서 리스트를 정렬하는 함수를 작성해줘.

# Answer
물론입니다! 파이썬에서 리스트를 정렬하는 방법은 여러 가지가 있지만, 가장 기본적인 방법으로 `sorted()` 함수와 `list.sort()` 메서드를 사용할 수 있습니다. 아래에 각각의 예시를 드릴게요.

---

### 1. `sorted()` 함수 사용하기

`sorted()` 함수는 원본 리스트를 변경하지 않고, 정렬된 새로운 리스트를 반환합니다.

```python
# 원본 리스트
my_list = [3, 1, 4, 1, 5, 9, 2, 6]

# 정렬된 새로운 리스트 생성
sorted_list = sorted(my_list)

print("원본 리스트:", my_list)
print("정렬된 리스트:", sorted_list)
```

**출력:**
```
원본 리스트: [3, 1, 4, 1, 5, 9, 2, 6]
정렬된 리스트: [1, 1, 2, 3, 4, 5, 6, 9]
```

---

### 2. `list.sort()` 메서드 사용하기

`list.sort()` 메서드는 원본 리스트를 **직접 수정**합니다.

```python
# 원본 리스트
my_list = [3, 1, 4, 1, 5, 9, 2, 6]

# 원본 리스트를 정렬
my_list.sort()

print("정렬된 리스트:", my_list)
```

**출력:**
```
정렬된 리스트: [1, 1, 2, 3, 4, 5, 6, 9]
```

---

### 3. 내림차순 정렬

기본적으로 오름차순으로 정렬되지만, `reverse=True` 옵션을 사용하면 내림차순으로 정렬할 수 있습니다.

```python
my_list = [3, 1, 4, 1, 5, 9, 2, 6]

# 내림차순 정렬
sorted_desc = sorted(my_list, reverse=True)
my_list.sort(reverse=True)

print("내림차순 정렬 (sorted):", sorted_desc)
print("내림차순 정렬 (sort):", my_list)
```

**출력:**
```
내림차순 정렬 (sorted): [9, 6, 5, 4, 3, 2, 1, 1]
내림차순 정렬 (sort): [9, 6, 5, 4, 3, 2, 1, 1]
```

---

### 4. 문자열 리스트 정렬

문자열 리스트도 동일하게 정렬할 수 있습니다.

```python
fruits = ["banana", "apple", "cherry", "date"]

# 오름차순 정렬
sorted_fruits = sorted(fruits)
fruits.sort()

print("정렬된 과일 리스트 (sorted):", sorted_fruits)
print("정렬된 과일 리스트 (sort):", fruits)
```

**출력:**
```
정렬된 과일 리스트 (sorted): ['apple', 'banana', 'cherry', 'date']
정렬된 과일 리스트 (sort): ['apple', 'banana', 'cherry', 'date']
```
